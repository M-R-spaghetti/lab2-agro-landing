Bachelor's Thesis

Information System for Communication and Informing Clients of an Agricultural Company

Інформаційна система комунікації та інформування клієнтів агрофірми

University: Sumy State University

PHP MySQL Web App Agribusiness Client-Server

Topic Relevance

In the context of digital transformation of Ukraine's agricultural sector, there is an urgent need to create effective communication tools between agricultural companies and their clients. Traditional methods of informing — phone calls, printed catalogs, personal meetings — no longer meet modern requirements for speed and convenience.

The development of a specialized information system will automate communication processes, ensure timely informing of clients about available products, prices, promotions and company news, as well as organize feedback to improve service quality.

Keywords

website, web application, agricultural company, information software, communication, PHP, MySQL, client-server architecture

Goals and Objectives

Research Objective

Development of an information system for communication and informing clients of an agricultural company.

Main Tasks

  1. Analysis of the subject area
  2. Determination of system requirements
  3. Selection of implementation methods and technologies
  4. Development of the web application
  5. Implementation of the feedback system
  6. Testing of system functionality

Research Methodology

The client-server architecture was chosen for the implementation of the information system, providing flexibility, scalability, and ease of use.

PHP

Server-side programming language for business logic processing and database interaction

MySQL

Relational database management system for storing information about clients and products

HTML/CSS/JS

Client-side technologies for creating an adaptive and interactive user interface

Client-Server Architecture

Separation of the system into client and server parts for optimal performance

Expected Results

As a result of the bachelor's thesis, a fully functional information system is expected, which will include:

  • Web Application — an interactive agricultural company website with product catalog, news system, and promotions
  • Feedback System — feedback forms, online consultations, and a messaging system for clients
  • Admin Panel — tools for content management, client database, and analytics
  • Database — a structured data storage system for products, clients, and communications
